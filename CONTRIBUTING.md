# Contributing to TransitionFX

## Bug Reports

If you encounter a bug, please report it via the [GitHub Issues](https://github.com/EmbarrassingMoment/TransitionFX/issues) page using the **Bug Report** template.

When reporting, include:

- Unreal Engine version and OS
- Steps to reproduce
- Expected vs. actual behavior
- Screenshots or videos if applicable

## Other Contributions

At this time, **pull requests and feature requests are not accepted.**

If you have a suggestion or question, please open an issue and describe it — but note that implementation requests may be closed without action.
